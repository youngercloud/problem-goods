针对等式
```
x   y   x + y
- + - = -----
u   v   u + v
```

首先我们为了方便计算，可以给等式左右两边去掉分母
- 给等式左边去掉分母，需要左右同乘 `u` 和 `v`
- 给等式右边去掉分母，需要左右同乘 `u + v`

所以等式在左右同乘 `u * v * (u + v)` 后变成了
```
x ∗ v ∗ (u + v) + y ∗ u ∗ (u + v) = (x + y) * u * v
```

化简
```
--> xvu + x * pow(v, 2) + yuv + y * pow(u, 2) = xuv + yuv

--> x * pow(v, 2) + y * pow(u, 2) = 0

--> x = -u^2
--> y = -v^2
```
