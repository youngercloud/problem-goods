根据题意可得，我们在一个 3 * 2 的矩阵中摆放形状时，只有 2 种情况。

![state](../../assets/1182A.png)

所以针对一个 `3 * n` 的矩阵，如果 `n` 为单数，我们可以输出 `0`

如果 `n` 为双数，针对每一个 `n`，我们都有 `dp[n - 2] * 2` 种方法来摆放。

```

    dp[2] = 2;

    }
    show(dp[n]);
}
```
